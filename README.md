🚀 Creovator: Agentic AI Event Automation SaaS
Creovator is an advanced AI-powered event organization platform designed to streamline the complexities of university and corporate event management. By integrating intelligent automation with a centralized dashboard, Creovator transforms how organizers plan, execute, and design their events.

💡 The Project Idea
The core vision of Creovator is to eliminate manual overhead in event coordination. It provides a chat-driven, agentic workflow that allows organizers to manage participants and generate professional event materials—such as posters and certificates—automatically. By auto-injecting user data into intelligent, built-in templates, Creovator ensures that branding and communication are consistent, efficient, and professional.

🛠️ Tech Stack
This project is built using a modern, high-performance stack:

Frontend Framework: React with Vite

Styling & UI: Tailwind CSS and Shadcn/UI

Backend & Database: Supabase (Real-time Database and secure Authentication)

Development & Design Tools:

Cursor IDE

Canva (Design logic and templates)

GitHub (Version control)

Testing: Playwright and Vitest

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine:

1. **Clone the Repository**
   Copy the project to your local system using:
   ```bash
   git clone [https://github.com/officiallymaya7/Final_Year_Project_FYP.git](https://github.com/officiallymaya7/Final_Year_Project_FYP.git)
Navigate to Project Directory
Go into the project folder:

Bash
cd Final_Year_Project_FYP
Install Dependencies
Install all the required packages using npm:

Bash
npm install
Environment Setup
Create a .env file in the root directory and add your Supabase credentials:

Code snippet
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
Run the Application
Start the development server:

Bash
npm run dev
